3. 自由设置

   ```
   文件扩展名,隐藏的文件
   win > 设置 > 个性化 > 主题 添加桌面图标
   win > 设置 > 个性化 > 开始 设置磁贴
   win > 设置 > 更新和安全  自由设置
   ```
4. 分区
   * win上软件许多都会默认占用盘符主目录，最好新建磁盘分区进行
   * 此电脑右击>管理>磁盘管理>选中分区右键>压缩卷
   * 输入压缩的量，MB单位(例如102400MB,就是10GB)，数字不会超过`可用压缩空间大小`
   ```
   注：操作系统是二进制，厂商标的是10进制，所以500g在电脑里显示465g
   500*1000^3^ / 1024^3^ = 465.66(GB)
   ```
