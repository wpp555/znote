3. 自由设置
   ```
   文件扩展名,隐藏的文件
   win > 设置 > 个性化 > 主题 添加桌面图标
   win > 设置 > 个性化 > 开始 设置磁贴
   win > 设置 > 更新和安全  自由设置
   ```
4. 分区(个人意愿)

   ```
   注：操作系统是二进制，厂商标的是10进制，所以500g在电脑里显示465g
   500*1000^3^ / 1024^3^ = 465.66
   ```
